          Sistema de Gestión Negocio Pro v2

          © 2024 - Todos los derechos reservados
          🗄️ Base de Datos Local (.db)
          📊 Excel / CSV

          Formato soportado: Hojas Compras, Ventas, Caja, Socios, Configuracion. Las ventas y compras del mismo día se procesan cronológicamente.

          ☁️ Google Drive

          Requiere configurar un Client ID de Google Cloud Console. Ve a la pestaña ⚙️ Configuración para ingresar tu Client ID.

          📱 Transfermóvil (para QR de cobro)
          Número de la tarjeta bancaria (16 dígitos numéricos).
          Celular de confirmación (8 dígitos, sin +53).
          📱 EnZona (para QR de cobro)
          Este es el texto/URL contenido en tu QR fijo de EnZona. Lo encontrarás en la app EnZona → Mi código QR.
          ☁️ Google Drive
          Obtén tu Client ID en Google Cloud Console → APIs y servicios → Credenciales. Ver instrucciones en la pestaña Ayuda.
          📘 Cómo usar el sistema

          📦 Inventario: Muestra el stock, precio promedio, precio de referencia y ganancia potencial.

          🛒 Comprar: Si el saldo es insuficiente, se te preguntará si registrar la compra como Aporte de capital.

          💵 Vender: Agrega productos al carrito, elige el método de pago. Si es Transfermóvil o EnZona, se genera un QR y debes confirmar el pago antes de registrar.

          💼 Caja: La Apertura de Capital solo se puede registrar una vez. Incluye nuevos tipos: Aporte, Venta en efectivo, Transferencia recibida.

          👥 Socios: Total de porcentajes no puede superar 100%.

          📊 Dashboard:

          • Caja actual: saldo total (efectivo + transferencias)
          • Efectivo en caja: solo ventas en efectivo
          • Transferencias: suma de transferencias recibidas

          📊 Formato Excel para importar:

          • Hoja: Compras — Fecha, Producto, Cantidad, PrecioUnitario
          • Hoja: Ventas — Fecha, Producto, Cantidad, PrecioVenta, MetodoPago (opcional)
          • Hoja: Caja — Fecha, Tipo, Concepto, Socio, Entrada, Salida
          • Hoja: Socios — Nombre, Porcentaje (opcional)
          • Hoja: Configuracion — Clave, Valor (opcional)

          ☁️ Google Drive — Cómo obtener el Client ID:

          1. Ve a console.cloud.google.com
          2. Crea un proyecto nuevo o selecciona uno existente
          3. Ve a APIs y servicios → Biblioteca y habilita Google Drive API
          4. Ve a APIs y servicios → Credenciales → Crear credenciales → ID de cliente OAuth 2.0
          5. Tipo de aplicación: Aplicación web
          6. En "Orígenes de JavaScript autorizados" agrega la URL donde usas este sistema (ej: file:// o tu dominio)
          7. Copia el Client ID y pégalo en la pestaña ⚙️ Configuración
          8. En "Pantalla de consentimiento OAuth" agrega tu email como usuario de prueba
